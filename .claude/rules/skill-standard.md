# Skill Standard

When creating or editing any file inside `skills/`, always read `skills/skills-creator/SKILL.md` before proceeding. That file defines the Eleven Brands skill standard — the mandatory structure, naming conventions, YAML frontmatter rules, and anti-patterns that every skill in this repository must follow.

Do not create or modify a skill without consulting it first.
